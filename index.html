Estas viendo un perrito!


La foto de abajo es actualizada cada minuto de forma aleatoria utilizando el API y un Cron Job.

La dirección de la API es:

https://dog.ceo/api/breeds/image/random

Código de Python utilizado:


    
    import os
    import json
    import requests
    
    #resultado = {}
    resultado = os.popen("curl https://dog.ceo/api/breeds/image/random 2>/dev/null").read()
    diccionario = json.loads(resultado)
    url = diccionario["message"].replace('\\', '')
    r = requests.get(url, allow_redirects=True)
    open('/var/www/html/perrito.jpg', 'wb').write(r.content)
